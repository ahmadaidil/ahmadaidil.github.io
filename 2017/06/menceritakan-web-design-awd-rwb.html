Pemikiran dan pendapat saya tentang web design

Adaptive Website Design (AWD)?

AWD adalah teknik web desain dimana konten didalamnya menyesuaikan/beradaptasi dengan tampilan perangkat yang digunakan untuk mengakses website tersebut.

Responsive Website Desgin (RWD)?

RWD adalah teknik web desain dimana tampilannya sudah disesuaikan dengan perangkat yang digunakan untuk mengakses website tersebut.

Smartphone dan tablet mengubah desain web secara signifikan?

Ya, karena smartphone dan tablet adalah perangkat yang paling banyak digunakan untuk mengakses web atau internet saat ini.

Penggunaan website untuk orang yang memiliki kekurangan fisik? Solusinya?

Tergantung fungsi atau kegunaan website yang ingin dibuat, jika memang target website tersebut untuk kalangan yang memiliki kekurangan fisik maka website tersebut harus memiliki tampilan yang cocok untuk mereka.

Ide atau konsep saat mempelajari desain web?

selain mempelajari teknik perancangan atau design, perlu juga mempelajari User Experience (UX) bagaimana pengalaman pengguna dalam mengakses website yang kita rancang.
